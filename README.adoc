== `pero chat` cli.

have your `.env` file on the root of the project.

```
OPENAI_API_KEY=sk-################################################
```

```
pero chat

chat endpoints

Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -f, --file     file to read from                                      [string]
  -r, --include-reasoning  include reasoning tokens if available       [boolean]
  -R, --reasoning-effort   low|medium|high (reasoning-capable models)  [string]
```

=== Tools configuration lookup

- If -t/--tools is provided, those files are loaded (in order).
- Otherwise, an automatic tools file is selected in this order:
  1) ./tools.yaml in current working directory
  2) ./tools.yml in current working directory
  3) bundled default tools.yaml
- Explicit -t files are merged after the automatic file and override on name conflicts.
- Use --omit-tools to disable the automatic tools file (only explicit -t files will be used).
- --omit-default-tools is deprecated; use --omit-tools instead.
